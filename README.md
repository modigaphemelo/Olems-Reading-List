# 📚 Olem's Reading List

A personal book tracking and review website. This project showcases books I've read with ratings, categories, and detailed reviews.

## ✨ Features

- 📖 **Book Catalog**: Display books with cover images, ratings, and metadata
- 🏷️ **Category Organization**: Books organized by genre (Nonfiction & Fiction)
- ⭐ **Rating System**: 1-10 rating scale with color-coded badges
- 📱 **Responsive Design**: Works on desktop, tablet, and mobile
- 🎨 **Clean UI**: Clean light theme with subtle shadows and smooth animations
- 🔍 **Easy Navigation**: Slide-out sidebar with category browsing
- 📅 **Reading Timeline**: Track reading dates

## 📱 Responsive Breakpoints

- **Desktop** (> 1024px): Full sidebar, 3-4 column grid
- **Tablet** (768px - 1024px): 2-3 column grid
- **Mobile** (< 768px): 1-2 column grid, mobile menu

## 🎨 Design Principles

- **Minimalism**: Clean, uncluttered interface with ample whitespace
- **Readability**: High contrast text, comfortable line height (1.6), good typography
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation
- **Performance**: Optimized images, lazy loading, minimal dependencies
- **Consistency**: Uniform spacing (using CSS custom properties), consistent patterns

## 📝 Code Style

- **HTML**: Semantic elements, ARIA labels where needed
- **CSS**: CSS Custom Properties for theming, consistent spacing scale
- **JavaScript**: Vanilla ES6+, modular functions
- **Images**: Lazy loading for performance
- **Icons**: Unicode emojis for simplicity and no external dependencies
